# 🧙‍♂️ 수리 마법사 (Math Wizard) : 이과생 수식이

**"수포자 출입 금지... 이과생을 위한 완벽한 공간!"**

시간을 멈추고 거대한 칠판을 열어 암산 문제를 푸세요! 정답을 맞히면 강력한 마법 투사체가 발사되어 몰려오는 적들을 박살 내는 꿀잼 타격감의 **2D 탑다운 액션 게임**입니다.

## 🎮 게임 특징 (Features)

* **뇌용량(마나) 시스템**: 시간은 금, 마나는 뇌용량! 가만히 있어도 뇌용량이 회복되지만, 적이 드롭하는 '당분 포션'을 자석처럼 쏙 빨아들여 파밍하면 순식간에 차오릅니다.
* **타임스탑 수리 영창 캐스팅**: 스페이스바나 `+,- 공격` 버튼을 누르면 시간이 멈추고(Slow Motion) 칠판이 펼쳐집니다! 계산 문제를 맞혀야 강력한 수식의 힘(투사체)이 발사됩니다.
* **오답 지진 페널티**: 만약 오답을 입력하면 칠판 전체가 부르르 떨리는 끔찍한 굴욕(?)을 맛보게 되며, 즉시 시간이 정상 속도로 회복되어 몬스터들이 순식간에 포위망을 좁혀옵니다!
* **압도적인 매트릭스 감성**: 무려 400개가 넘는 빽빽하고 요동치는 수학/물리 테마의 배경과 수식들이 스플래시 화면과 타이틀 화면을 장식하여 완벽한 '이과생 칠판 감성'을 선사합니다.
* **체력(HP)과 서바이벌 디펜스**: 피격당하면 체력이 10씩 깎이고 화면이 시뻘겋게 번쩍입니다! 사방에서 3초마다 끝없이 나타나는 빨간 적들로부터 최대한 살아남으세요!
