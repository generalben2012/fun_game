- [x] 2. 기본 디렉토리 구조(Scenes, Scripts, Resources, Assets) 정리.
- [x] 3. 플레이어 캐릭터(이수식) Scene 생성 및 WASD 8방향 이동 스크립트(CharacterBody2D) 구현.
- [x] 4. 메인 뷰포트(카메라) 세팅 및 카툰 렌더링 스타일의 기본 타일맵 테스트 환경 구성.

## Phase 3. 전투 액션 및 스킬 연출 (챕터 1 기준)
