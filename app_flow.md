# 수리 마법사 (가제) - 게임 실행 흐름도 (Game Flow)

본 문서는 플레이어가 게임을 켜서 끌 때까지 경험하게 되는 화면(Scene) 전환 흐름과 각 화면의 주요 기능을 정의합니다.

## 1. 앱 시작 및 초기 화면
1. **스플래시 스크린 (Splash Screen)**
   - 게임 로고 및 개발사 로고 노출 (약 2초)
   - 뒷배경으로 알 수 없는 수학 기호들이 비처럼 쏟아지는 연출 (매트릭스 패러디)
2. **타이틀 화면 (Title & Main Menu)**
   - **게임 시작 (Start / Continue)**: 저장된 스테이지부터 바로 시작하거나 스테이지 선택 화면으로 이동.
   - **오답 노트 (Study / Collection)**: 지금까지 틀린 문제나 해금한 마법(수학 공식) 도감을 확인.
   - **설정 (Settings)**: BGM/BGS 볼륨 조절.
   - **종료 (Exit)**: 앱 종료 여부 묻는 팝업.

## 2. 게임 진입 및 준비
3. **스테이지 선택 화면 (World Map)**
   - **월드 맵**: 챕터 1~3 전체 지형(학교 -> 교육청 -> 교육부)을 보여주며, 클리어한 곳까지 길이 열려있음.
   - **스테이지 정보**: 선택한 스테이지의 미션 내용, 등장하는 적(보스), 드롭되는 재화('이과의 혼') 및 당분(마나 물약) 정보 표시.
   - **스테이지 진입 버튼**: 누르면 로딩 후 전투 화면으로 진입.
4. **수리 영창 셋업 (상점 및 스킬 장착)** 
   - *스테이지 선택 화면 내 메뉴 또는 독립된 화면*
   - 전투 전 수집한 '이과의 혼(경험치)'을 소모해 **뇌용량(최대 마나)**을 늘리거나.
   - 보유한 당분(아이템)을 장착.
   - 퀵슬롯에 사용할 마법(덧셈, 구구단, 이차함수 등)을 배치.

## 3. 핵심 플레이 루프 (전투)
5. **전투 화면 (Battle Stage)**
   - 기본적으로 탑다운(Top-down) 뷰에서 키보드(WASD)를 이용해 8방향으로 자유롭게 이동.
   - 몹과 조우 시 화면 전환 없이 캐릭터 대기.
   - **공격 버튼(수리 영창)**: 누르는 순간 BGM이 멈추거나 긴박하게 바뀌고 **'시간 정지'** 연출. 
     - 칠판 UI(수리 영창 패널)가 위에서 아래로 쾅! 떨어짐.
   - **문제 풀이 (입력)**: 주관식 키패드 또는 4지선다 객관식 버튼으로 답안 입력. (입력 시간 동안 뇌용량 서서히 감소)
   - **액션 연출**: 정답 시 마법이 시원하게 나가며 적 타격. 오답 시 분필이 부러지는 듯한 소리와 함께 캐스팅 취소/뇌용량 페널티.
6. **일시 정지 메뉴 (Pause Popup)**
   - 전투 중 우측 상단 톱니바퀴 클릭 시 시간이 정지하며 팝업 등장.
   - **메뉴 항목**: [계속하기 (Resume)], [설정 (Settings)], [전투 포기 (Give Up: 월드맵으로 귀환)]

## 4. 결과 및 종료
7. **스테이지 결과 화면 (Result Screen)**
   - **클리어 시**: "Q.E.D. (증명 완료)" 도장이 크게 찍히며, 획득한 경험치, 콤보 수, 클리어 타임 등급(S, A, B 등) 보상 출력.
   - **실패 시 (뇌용량 0 또는 체력 0)**: "계산 오류 (Syntax Error)" 메시지와 함께 게임 오버.
   - [다시 하기] / [월드 맵으로] 버튼 제공.

## 5. 게임 종료 프로세스
8. **게임 종료 확인 팝업**
   - 타이틀 화면이나 월드맵에서 'ESC' 키 또는 종료 버튼 입력 시 출력.
   - "진짜 수학 공부하러 가시나요? (게임 종료하시겠습니까?)" [예/아니오]
